Instructions

  1. Enter the employee name and department as it appears "Employee/Department"
  2. Set the period ending to the Saturday the travel occurred. This will populate the period end date and correctly populate the dates from Sunday to Saturday on the Excel spreadsheet, a tedious time saver function.
  3. If the expense report includes travel , check the checkbox next to travel. Select the start and end travel dates. This will populate the breakfast and lunch per diem, excluding breakfast for day one and dinner for the final day of travel.
  4. Once the Excel spreadsheet is created, go to "page-layout", then "page set-up" and select "fit to one page". This is very important formatting step for printing the document efficently.
  5. Also, uncollapse the Employee/Department, and the Trip Purpose columns prior to printing for signing and scanning.
  6. Make sure to adjustment to the Excel spreadsheet as needed post creation to ensure the document is accurate.

Please note that this application generates an Excel document based on your form inputs. Some browsers may block the automatic download of files as a security measure. If you're not seeing the Excel file download after submitting the form, please check your browser settings to ensure that file downloads are allowed. Additionally, make sure that pop-ups are not blocked, as the download may open in a new window or tab. If you're still experiencing issues, try using a different browser or disabling any browser extensions that may interfere with file downloads. Remember to always use the latest version of your browser for the best experience.
