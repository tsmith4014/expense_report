Instructions

  1. Enter the employee name and department as it appears "Employee/Department"
  2. Set the period ending to the Saturday the travel occurred. This will populate the period end date and correctly populate the dates from Sunday to Saturday on the Excel spreadsheet, a tedious time saver.
  3. If the expense report includes travel , check the checkbox next to travel. Select the start and end travel dates. This will populate the breakfast and lunch per diem, excluding breakfast for day one and dinner for the final day of travel.
  4. Make sure to adjustment to the Excel spreadsheet as needed post creation to ensure the document is accurate.

Please note that this application generates an Excel document based on your form inputs. Some browsers may block the automatic download of files as a security measure. If you're not seeing the Excel file download after submitting the form, please check your browser settings to ensure that file downloads are allowed.
